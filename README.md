
# 1VD-FTV Intelligent Diagnostic Platform

Fullstack starter platform for:
- Live stream analysis
- DTC analysis
- Smart diagnostics
- Toyota 1VD-FTV engine support

## Backend

### Run

cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload

## Frontend

cd frontend

npm install

npm start

## Features

- FastAPI backend
- React frontend
- Smart rail pressure analysis
- Turbo underboost detection
- AI-ready architecture
